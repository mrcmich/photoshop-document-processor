Class: FiltroTonalizzazioneCMYK

FiltroTonalizzazioneCMYK(filtroLetturaTonoCMYK)

new FiltroTonalizzazioneCMYK(filtroLetturaTonoCMYK)

Constructor function per la creazione di un filtro tonalizzazione CMYK, oggetto che si occupa della tonalizzazione di un set di documenti ad un tono di riferimento specificato dall'utente, collaborando con un filtro lettura tono cui demanda la rilevazione e validazione del tono dei documenti e la loro organizzazione in una tabella dei toni. Ha FiltroTonalizzazioneAstratto come prototipo.
Parameters:
Name Type Description
filtroLetturaTonoCMYK FiltroLetturaTonoAstratto il filtro lettura tono da associare al filtro tonalizzazione.

_determinaRiferimentoCanale(canale, riferimentoDefault) → {Array}

Metodo per la richiesta del riferimento per uno specifico canale all'utente, sotto forma di un un array contenente un singolo valore percentuale (se l'utente inserisce un valore) oppure due valori percentuali (nel caso in cui l'utente inserisca un range del tipo [min, max], i due valori ne identificano gli estremi). Il riferimento di default passato come parametro viene usato per inizializzare il campo di testo in cui l'utente inserirà il riferimento per il canale. Ritorna l'array di cui sopra, o undefined se l'utente ha annullato l'inserimento. Se l'input dell'utente non è valido, all'utente viene mostrato un avvertimento e richiesto di inserire nuovamente il riferimento.
Parameters:
Name Type Description
canale string il nome del canale di cui si richiede il riferimento, come mostrato all'utente.
riferimentoDefault number il riferimento di default per il canale, usato per inizializzare il relativo campo di testo.

_determinaRiferimentoEffettivoCanale(riferimento, valoreIniziale) → {number}

Metodo per la determinazione della quantità percentuale di riferimento effettiva di un canale, a partire da un riferimento e dalla quantità percentuale iniziale del canale. Se riferimento contiene un singolo valore, tale quantità coincide con il valore. Se riferimento contiene due valori (che identificano gli estremi di un intervallo) tale quantità viene posta uguale al minimo dell'intervallo se la quantità iniziale è inferiore al minimo oppure al massimo se la quantità iniziale è superiore al massimo. Ritorna la quantità percentuale di riferimento effettiva.
Parameters:
Name Type Description
riferimento Array array contenente il riferimento per il canale, sotto forma di singolo valore o di coppia di valori (nel secondo caso, i due valori rappresentano gli estremi del range del tipo [min, max]).
valoreIniziale number quantità percentuale iniziale del canale, usata per valutare la quantità percentuale di riferimento effettiva.

_determinaTonoRiferimento(tonoDefault) → {Object}

Metodo per la richiesta del tono di riferimento all'utente, sotto forma di una lista di riferimenti (uno per ciascun canale CMYK) che possono essere costituiti da singoli valori oppure da range del tipo [min, max]. Il tono di default passato come parametro viene usato per inizializzare i campi di testo in cui l'utente inserirà poi i riferimenti.
